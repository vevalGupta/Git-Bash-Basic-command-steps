## üß© Introduction

**Git** is a distributed version control system that tracks changes in source code during software development.  
**Git Bash** provides a Unix-style terminal on Windows to execute Git commands smoothly.

This repository is designed for:
- Students and beginners learning version control.
- Developers setting up Git for the first time.
- Anyone who wants a simple, command-based Git workflow reference.

---

## üíª What is Git Bash?

**Git Bash** is an application for Microsoft Windows that provides:
- **Git command-line features**
- **Bash emulation** (Linux terminal experience)
- **Shell scripting capabilities**

It allows users to use both **Git** and **Unix commands** seamlessly.
