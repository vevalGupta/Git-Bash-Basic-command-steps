# Git Basics - Reference Guide

This repository contains basic Git commands and steps to help beginners understand how to work with Git and GitHub. Git is a version control system used to track changes in code and collaborate with others.

---

## What is Git?

Git is a **version control system** that helps developers **track changes**, **revert to previous versions**, and **collaborate** with others easily. It allows multiple people to work on the same project without overwriting each other's work.

---

## What is GitHub?

**GitHub** is a cloud platform where you can **store Git repositories online**, share them with others, and collaborate on projects.

---

## Basic Git Commands

| Command | Description |
|--------|-------------|
| `git init` | Creates a new local Git repository |
| `git clone <url>` | Copies a remote repository to your computer |
| `git status` | Shows the current status of your files |
| `git add .` | Stages all changed files for commit |
| `git commit -m "message"` | Saves the staged changes with a message |
| `git branch` | Shows all branches |
| `git checkout <branch>` | Switches to another branch |
| `git push origin <branch>` | Sends your commits to GitHub |
| `git pull origin <branch>` | Downloads latest changes from GitHub |
